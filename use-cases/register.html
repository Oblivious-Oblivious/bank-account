Register
Data:
  • <Username>
  • <PIN>
  • <Password>
Primary Course:
  1. User issues "Register" command with above data.
  2. System validates all data for duplication.
  3. User registers the user into the saver.
  4. System restarts the application.
Duplication Error (2)
  1. System delivers error message to user; Username already exists.
  2. System repeats "Register"
